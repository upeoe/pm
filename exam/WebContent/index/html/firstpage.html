  • Clustering analysis(聚类分析)

    Cluster analysis refers to the analysis of multiple classes of objects that are composed of a collection of physical or abstract objects. The goal of clustering analysis is to gather data to classify the data. Clustering stems from many areas, including mathematics, computer science, statistics, biology and economics. In different application fields, many clustering techniques have been developed. These techniques are used to describe the data and measure the similarity between different data sources.
